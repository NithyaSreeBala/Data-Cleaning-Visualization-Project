# Data Cleaning & Visualization Project

## 📌 Project Objective
The goal of this project is to clean, process, and visualize raw data using Python libraries like Pandas, Matplotlib, and Seaborn.

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Dataset
Titanic Dataset

---

## 🔍 Tasks Performed
- Loaded raw dataset
- Handled missing values
- Removed unnecessary columns
- Cleaned dataset
- Created visualizations
- Extracted insights from data

---

## 📊 Visualizations Created
- Survival Count
- Gender vs Survival
- Age Distribution
- Passenger Class Count
- Correlation Heatmap

---

## 📖 Key Insights
- Females had higher survival rates
- Most passengers were between age 20–40
- Majority of passengers traveled in 3rd class
- Passenger class influenced survival chances
